# 1389A - LCM Problem

**Enlace:** https://codeforces.com/problemset/problem/1389/A

## Resumen del problema
Dado un rango de enteros `[L, R]`, se pide encontrar dos enteros `x` y `y` tales que `L <= x < y <= R` y el mínimo común múltiplo (LCM) de `x` e `y`, `lcm(x, y)`, también sea menor o igual a `R`. Si existe tal par, se debe imprimir `x` e `y`. Si no existe ningún par que cumpla estas condiciones, se debe imprimir `-1 -1`.

## Input
-   La primera línea contiene un solo entero `t` ($1 \le t \le 1000$), el número de casos de prueba.
-   Para cada caso de prueba:
    -   La única línea contiene dos enteros `L` y `R` ($1 \le L \le R \le 10^9$).

## Output
Para cada caso de prueba, imprime dos enteros `x` e `y` separados por un espacio que satisfagan las condiciones, o `-1 -1` si no existe tal par.

## Restricciones

| Variable     | Rango      | Notas                               |
| :----------- | :--------- | :---------------------------------- |
| t (casos)    | [1, 1000]  |                                     |
| L, R (límites) | [1, 10^9]  | Límite inferior y superior del rango |

## Análisis Matemático
El problema pide encontrar dos enteros $x$ y $y$ en el rango $[L, R]$ tales que $L \le x < y \le R$ y $\text{lcm}(x, y) \le R$.

Para minimizar el $\text{lcm}(x, y)$, es ventajoso elegir $y$ como un múltiplo de $x$.
Consideremos el caso más simple donde $x = L$.
El múltiplo más pequeño de $L$ que es mayor que $L$ es $2L$.
Si elegimos $y = 2L$, entonces $\text{lcm}(L, 2L) = 2L$.

Para que el par $(L, 2L)$ sea una solución válida, debe cumplir las siguientes condiciones:
1.  $L \le x < y \le R \implies L \le L < 2L \le R$. La condición $L < 2L$ es verdadera para $L \ge 1$. Por lo tanto, solo necesitamos $2L \le R$.
2.  $\text{lcm}(x, y) \le R \implies \text{lcm}(L, 2L) \le R \implies 2L \le R$.

Así, si $2L \le R$, el par $(L, 2L)$ es una solución válida.
Si $2L > R$, significa que incluso el múltiplo más pequeño de $L$ (que es $2L$) excede el límite superior $R$. En este escenario, no es posible encontrar un $y$ que sea múltiplo de $L$ y que esté dentro del rango $(L, R]$ tal que $\text{lcm}(L, y) \le R$. Para problemas de esta dificultad, esta observación es suficiente para concluir que no existe ninguna solución.

## Algoritmo
1.  Leer el número de casos de prueba `t`.
2.  Para cada caso de prueba:
    a.  Leer los enteros `L` y `R`.
    b.  Evaluar la condición:
        i.  Si `2 * L > R`, significa que no se puede formar el par `(L, 2L)` dentro del rango `[L, R]` y que `lcm(L, 2L)` excedería `R`. En este caso, imprime `-1 -1`.
        ii. Si `2 * L <= R`, el par `(L, 2L)` cumple todas las condiciones: `L <= L < 2L <= R` y `lcm(L, 2L) = 2L <= R`. En este caso, imprime `L 2 * L`.

## Notas Adicionales
*   La solución se basa en una observación simple y la propiedad del mínimo común múltiplo. Para un número $x$, el mínimo común múltiplo más pequeño con otro número $y > x$ es $2x$ (cuando $y=2x$).
*   El problema está diseñado para que esta solución simple sea suficiente. No es necesario buscar pares más complejos o realizar búsquedas exhaustivas.
